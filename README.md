# Skills

Claude Code 自定义 skill 集合。

## product-narrative / 产品叙事策略

产品叙事营销框架。基于百年营销实践与近十年媒介变革中提炼的变与不变的规律，为任意产品生成完整的叙事方案。

**三轴驱动**：意义绑定 / 问题-解决方案（负面情绪释放）/ 认知节奏控制

**五大操作维度**：稀缺性、身份叙事、问题-解决、起源故事、未来叙事

**覆盖全链路**：轴（底层机制）→ 模态（载体）→ 操作维度（策略）→ 媒介适配（执行），FAB 作为内容翻译器贯穿其中

**13个参照案例**：Supreme、茅台、lululemon、小红书、戴森、Keep、Patagonia、褚橙、Tesla、泡泡玛特、Apple、瑞幸、观夏、Notion、蜜雪冰城、李佳琦、雷军。案例作为模式匹配的参照系——用户回答五个引导问题后，agent 匹配最近似案例，提取底层规律，识别当前变化条件，在巨人肩膀上向前创新。

内置窗口-内容适配表（3秒/15秒/60秒/长内容）、度量指标、失败模式清单和七步使用流程。

触发：`/product-narrative`

## skill-meta-designer / Skill 元提示设计师

将自然语言描述转化为结构规范的 SKILL.md 文件。

**两阶段交付**：先输出结构大纲确认方向，再生成完整 SKILL.md

**三种意图类型适配**：流水线型（步骤驱动）、认知模式型（角色+判断原则）、混合型

**完整模块体系**：触发条件、角色定义、输入规范、初始化、处理流程、自治边界、交互格式、约束与反模式、输出规范、协作接口、失败处理、示例

内置 Markdown 语法使用规则（最大化 token 效率）、模块建议矩阵、15 项质量自查清单。优先适配 Claude skill 体系，兼容其他 LLM agent。

触发：`/skill-meta-designer`
