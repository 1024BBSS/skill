# 问题-解决叙事案例

## 案例1：戴森 — 建构你从未意识到的问题

**轴定位**
- 轴1 意义绑定：绑价值观（戴森=不将就/科技改变生活）+ 绑身份（用戴森的人=对生活品质有要求）
- 轴2 情绪路径：负向驱动。先让你意识到"原来我一直忍受的东西是个问题"→ 焦虑 → 戴森解决 → 释放
- 轴3 认知节奏：强张力开场（放大问题）→ 技术揭秘（张力加深——"你不知道的真相"）→ 产品演示（释放）

**FAB**
- F：数码马达、气旋分离技术、HEPA滤网
- A：吸力不衰减（vs传统吸尘器越用越弱）
- B：你的家并没有你以为的那么干净——现在可以了

**操作维度**
- 问题-解决为主：戴森最强的能力是**建构问题**——在戴森之前，没人觉得吹风机是个问题
- 起源故事为辅：James Dyson 5127次原型失败的故事 = 创始人使命的经典案例

**FAB × 轴2 的精确运作**
1. 问题建构："你知道你的床上有多少螨虫吗？"（F层面的信息制造认知冲击）
2. 负面情绪积累："传统吸尘器只是把灰尘搅起来又吹回空气中"（A层面的对比加深焦虑）
3. 解决方案："戴森气旋技术，分离99.97%的微尘"（F→A→B完整释放）

**具身**
- James Dyson本人：白发、工程师气质 → 面孔可信度极高（老=可信）
- 关系姿态：引领型——"我知道一些你不知道的事"
- 产品演示视频中无人出镜时，产品本身成为"技术具身"（马达转速的声音=音频具身）

**媒介适配**
- 3秒："你的枕头里有200万只螨虫"（纯B点冲击，具身面孔可省略）
- 15秒：螨虫事实 → 戴森吸出来的灰尘量对比（B→A）
- 长内容：James Dyson实验室纪录片（F→A→B + 起源故事完整链）

**度量**
- 搜索词变化："螨虫 吸尘器""头发 吹风机伤害" — 用户是否开始搜索戴森定义的问题
- 产品演示视频的完播率（认知节奏是否有效）

**启示**：戴森的核心竞争力不是技术，是**问题建构能力**。它先让你意识到一个你忽略的问题，再用技术解决它。产品解决的不是真实世界的问题，是叙事制造出来的问题。

## 案例2：Keep — 从焦虑到精确场景

**轴定位**
- 轴1 意义绑定：绑身份（自律的人用Keep）
- 轴2 情绪路径：负向驱动。"你是不是又胖了/又没运动" → 焦虑 → 打开Keep → 完成训练 → 释放
- 轴3 认知节奏：每次训练=一个完整的张力-释放周期（倒计时=张力，完成=释放，打卡=重置）

**FAB**
- F：课程体系、AI个性化推荐、社区
- A：比自己练有结构，比去健身房成本低
- B：你不需要意志力，你需要一个系统——有了系统你就是自律的人

**操作维度**
- 问题-解决：从普遍焦虑（"我应该运动"）到精确场景（"久坐办公族的10分钟肩颈放松"）
- 身份叙事："自律给我自由"——slogan本身就是身份标签

**FAB × 轴2 的精确运作**
1. 普遍焦虑入口："你已经30天没有运动了"（推送通知=问题建构）
2. 精确场景转化："今天试试这个15分钟燃脂——适合加班后"（场景精确到不可拒绝）
3. 情绪释放："训练完成！你今天超过了87%的用户"（社交比较+成就=双重释放）

**启示**：Keep示范了问题-解决叙事的两级火箭——用普遍焦虑获客（"你该运动了"），用精确场景转化（"这个课程正好适合你现在"）。焦虑是入口，场景是转化器。
