# 起源故事叙事案例

## 案例1：Patagonia — 创始人使命的终极形态

**轴定位**
- 轴1 意义绑定：绑价值观（Patagonia=环保主义不是口号，是行动）
- 轴2 情绪路径：负向+正向混合。负向="地球在被毁坏"→焦虑；正向="我的消费可以改变这件事"→意义感
- 轴3 认知节奏：低频深度叙事为主。不追求高频触达，每一次叙事都是长周期。

**FAB**
- F：再生材料、公平贸易认证、可修复设计
- A：别的品牌讲环保是营销，Patagonia把公司捐给了地球（2022年将公司所有权转让给环保信托）
- B：穿Patagonia=你的价值观有了物理载体

**操作维度**
- 起源故事为主：Yvon Chouinard从攀岩者到环保企业家的故事 = "我觉得消费不应该以牺牲地球为代价"
- 未来叙事为辅："1% for the Planet""Don't Buy This Jacket"——品牌在讲一个还没结束的故事

**起源故事的层级拆解**
1. 个人起源：Chouinard是攀岩者，在岩壁上亲眼看到自然被破坏（第一人称见证=不可伪造）
2. 产品起源：因为自己需要才开始做装备，不是为了赚钱（创始人使命="我觉得户外装备不应该伤害户外"）
3. 理念迁移：他的理念=我认同的理念。买Patagonia不是消费，是投票。

**具身**
- Chouinard本人：老、朴素、不像CEO → 面孔可信度极高
- 关系姿态：平视/同侪——他不是在卖东西，他是在邀请你加入一个运动
- "Don't Buy This Jacket"广告：品牌主动劝你不买 → 反直觉=强张力 → 信任=强释放

**媒介适配**
- 3秒：荒野画面+一句话（B："The planet is our only shareholder"）
- 长内容：纪录片《180° South》、Chouinard自传——起源故事的完整形态只适合长内容

**度量**
- 主动传播率：Patagonia的故事被消费者主动转述的频率
- 品牌联想测试：提到"环保"时Patagonia的无提示提及率

**启示**：Patagonia证明了起源故事的最强形态是创始人**真的在做**他说的事。当F（产品特征）、A（差异优势）、B（益处）和起源故事完全一致时，叙事的可信度无法被攻击。反例：一个创始人讲环保故事但坐私人飞机——起源故事的失败模式就是言行不一被扒皮。

## 案例2：褚橙 — 人生故事嫁接到产品

**轴定位**
- 轴1 意义绑定：绑情感（褚时健的跌宕人生→不服输的精神）
- 轴2 情绪路径：负向→正向转化。人生低谷的沉重→75岁重新创业的振奋→"这个橙子有灵魂"
- 轴3 认知节奏：强叙事弧——跌落（张力）→ 谷底（张力加深）→ 重新站起（释放）→ 橙子成了（重置为新起点）

**FAB**
- F：云南哀牢山、冰糖橙、24:1黄金甜酸比
- A：品质确实好，但真正的A是：这个橙子背后站着褚时健
- B：吃褚橙=致敬一种不服输的精神=我也是这样的人

**操作维度**
- 起源故事为绝对主导：没有褚时健的故事，这就是一个普通的云南冰糖橙
- 身份叙事为辅：买褚橙在当年几乎是企业家圈层的社交货币

**起源故事的叙事结构**
- 不是品牌包装出来的故事，是真实人生 → 不可复制、不可伪造
- "老=可信"的极致应用：75岁再创业，时间本身就是最强背书
- 信任迁移：褚时健的人格 → 橙子的品质。消费者的逻辑是"这个人做事不会差"

**具身**
- 褚时健本人：瘦、老、站在果园里 → 朴素面孔=高信任
- 关系姿态：不是引领也不是服务，而是一种**沉默的在场**——他不说服你，他的存在本身就是叙事

**启示**：褚橙证明了起源故事可以完全覆盖产品本身——消费者买的不是橙子，是一个人的精神。但这也是它的脆弱性：褚时健去世后，叙事引擎失去了核心具身，品牌面临"故事讲完了然后呢"的问题。起源故事如果没有未来叙事接棒，会随时间衰减。
